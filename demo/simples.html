Simples

Cria um objeto e um container. Ao arrastar o objeto até o container uma mensagem de alerta contendo os dados do objeto é exibida.

<div ng-controller="DemoController as ctrl"> <div class="block" rtm-drag-object="{name: 'foo'}">Drag</div> <div class="block" rtm-drag-container="list" rtm-ondrop="ctrl.eventAlert(event)">Drop</div> </div> app.controller('DemoController', function(DragInfo) { this.eventAlert = function(event) { alert(event.data.name); } }); #demo .block { display: inline-block; width: 100px; height: 100px; line-height: 100px; text-align: center; margin: 16px; border-radius: 4px; background-color: #222; color: #fff; }
